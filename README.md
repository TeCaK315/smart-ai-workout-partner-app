# Smart AI Workout Partner App

## Описание
Индивидуальный AI-тренер для повышения мотивации и эффективности тренировок.

## Проблема
Многие люди испытывают трудности с мотивацией во время тренировок.

## Решение
AI-партнер, который подбадривает и предлагает персонализированные тренировки.
